Deployment:

1. Install Java and Python3.
2. Download wkhtmltox from [https://github.com/wkhtmltopdf/wkhtmltopdf/releases/download/0.12.4/wkhtmltox-0.12.4_msvc2015-
win64.exe] and install it. After installation set environment path.
3. Run Libraries.bat as an administrator from /Resources folder.
4. Download ImageMagick from [https://www.imagemagick.org/script/download.php#windows] and install it for video. After installation set environment path 
like [C:\Program Files\ImageMagick-7.1.0-Q16-HDRI]

Usage:
1. Create a folder for each patient, this will store raw material of the patient i.e OrthoReport.pdf, IPR-1.pdf, TPR-1.pdf and OrthoReport.html.
2. IPR Download Process:
   Print -> In destination dropdown select "Microsoft Print To PDF" -> More Setting select A4 in paper size -> Download.
3. Create a folder named Videos and place all 5 videos in it.
3. Double click on Run Utility.

Results:
1. finalReport folder will have patient name wise folders containing the final report.
