Deployment:
1. Install java in ur pc.
2. Change source folder, temp folder and dest path in config.properties according to ur pc.
3. Change path in start.bat , same as done in step 2.

Usage:
1. Create a folder for each patient, this will store raw material of the patient i.e OrthoReport.pdf, IPR-1.pdf, TPR-1.pdf.
2. IPR Download Process:
   Print -> In destination dropdown select "Microsoft Print To PDF" -> More Setting select A4 in paper size -> Download.
3. Double click on Run Utility.

Results:
1. finalReport folder will have patient name wise folders containing the final report.
2. End user needs to verify if the report seems okay or not for rare exceptions.
